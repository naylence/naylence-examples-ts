Multi-Agent Text Analysis Example

This example demonstrates a distributed multi-agent pipeline where one agent orchestrates multiple others to analyze text. The analysis agent broadcasts requests to both a summarizer agent and a sentiment agent, then merges their results.

  • Analysis Agent orchestrates at analysis@fame.fabric
  • Sentiment Agent scores sentiment (1-5) at sentiment@fame.fabric
  • Summarizer Agent generates summaries at summarizer@fame.fabric
  • Scatter-gather pattern via Agent.broadcast runs agents in parallel
  • Envelope inspector shows outbound and inbound messages for this client
